# PDF Chatbot

A production-ready starter project structure for a PDF-based chatbot application.

## Project structure

- app.py: Streamlit entrypoint
- src/chatbot_app/: application package
- tests/: automated tests

## Run locally

```bash
./.venv/bin/python -m streamlit run app.py
```

## Development

```bash
./.venv/bin/python -m unittest discover -s tests
```
